# Clone do Facebook

Este projeto é uma recriação de uma das páginas do Facebook, desenvolvida com o objetivo de aprimorar habilidades em HTML e CSS, com ênfase em responsividade e na metodologia BEM (Block Element Modifier) para escrita de CSS.

##Tecnologias Utilizadas

- **HTML5**: Estruturação semântica do conteúdo.
- **CSS3**: Estilização e layout responsivo.
- **Metodologia BEM**: Organização e manutenção eficiente do código CSS.

## Estrutura do Projeto

- `index.html`: Arquivo principal contendo a estrutura HTML da página.
- `css/`: Diretório com os arquivos CSS para estilização.
- `img/`: Pasta que armazena as imagens utilizadas no projeto.
